Latest projects

ObraGo

Este projeto visa criar uma plataforma para reunir produtos de diferentes lojas. Inclui funcionalidades como cadastro e login para clientes, uma página inicial que exibe produtos de várias lojas, além de recursos completos de cadastro e login para lojas, incluindo operações CRUD para gerenciar seus produtos. Desenvolvido utilizando Java, Servlet, Hibernate, Apache TomCat e páginas .JSP.

FutBoleiro

FutBoleiro é um projeto abrangente que incorpora cadastro e login, conectado a um banco de dados MySQL. Além disso, apresenta dinamicamente notícias alimentadas por dados provenientes do banco de dados. A implementação do back-end é realizada usando Node.js e AJAX.

LabManager

LabManager é um projeto de gerenciamento completo (CRUD) de laboratórios e computadores. Foi desenvolvido em .NET e ASP.NET Core, proporcionando prática em banco de dados e back-end. O projeto envolve tecnologias como C#, MySQL e HTML.

Buscador de CEP

Este é um buscador simples de CEP que consome a API ViaCep. A aplicação, desenvolvida enquanto estudava React.js, retorna informações detalhadas, como rua, complemento, bairro, cidade e estado. Uma excelente introdução ao desenvolvimento em React.js e à integração com APIs externas.
